Explain the following code or system component. Structure your explanation as:

1. **Purpose** - What it does and why it exists
2. **How it works** - Key mechanisms, data flow, algorithms
3. **Dependencies** - What it relies on, external services, libraries
4. **Key decisions** - Non-obvious design choices and their rationale

Be concise but thorough. Use the codebase context to inform your explanation.
